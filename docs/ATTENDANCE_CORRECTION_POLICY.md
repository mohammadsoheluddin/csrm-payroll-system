# Attendance Correction Policy

# CSRM Payroll System

---

# PURPOSE

Defines:
attendance correction workflow.

---

# IMPORTANT PRINCIPLE

Attendance correction should remain:
controlled and auditable.

---

# COMMON CORRECTION TYPES

- missed punch
- manual correction
- holiday correction
- duty correction
- leave adjustment

---

# IMPORTANT RULE

After attendance finalization:
normal editing should stop.

---

# CORRECTION FLOW

Correction Request
↓
Review
↓
Approval
↓
Attendance Update
↓
Audit Entry

# FUTURE FEATURES

- correction approval
