# Payroll Audit Checklist

# CSRM Payroll System

---

# PURPOSE

Defines:
monthly payroll verification checklist.

---

# PRE-PROCESS CHECKS

- attendance finalized
- leave verified
- holiday verified
- movement verified
- salary structure verified

---

# SALARY SHEET CHECKS

- duplicate payroll prevention
- salary validation
- deduction validation
- employee count validation

---

# TIME BILL CHECKS

- OT hour validation
- OT eligibility validation
- tiffin validation

---

# STATEMENT CHECKS

- bank amount validation
- cash amount validation
- deduction validation

---

# PAYMENT CHECKS

- bank account validation
- routing validation
- payment summary validation

---

# IMPORTANT RULE

Payroll audit should happen BEFORE:
approval and payment.
