# CSRM Payroll System — Module Discovery Lock

Date: 2026-05-15

## Decision

The open-ended module discovery exercise is now locked.

The project explored hundreds of possible modules across:

```txt
Payroll
HR
Admin
Factory workforce
Attendance
Leave
Time bill / OT
Payments
Documents
Compliance
Facilities
Security
Transport
Recruitment
Learning
Performance
AI

```

The project will not continue searching for new modules randomly.

## Rule Going Forward

Every new idea must be classified as one of:

```txt
1. Current Payroll MVP
2. Payroll Enterprise Upgrade
3. HR/Admin Extension
4. Future Separate ERP/App
5. Research Archive Only
```

## Immediate Focus

```txt
Part-B53 — Employee Full Profile / Digital Service Book Backend Foundation
```

## Reason

Employee profile is the central foundation for future HR/payroll modules. It connects employee master data, salary, payment, attendance, leave, movement, documents, timeline, audit, and legacy salary archive without mixing external legacy data into native payroll calculation.
