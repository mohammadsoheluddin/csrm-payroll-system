# Salary Structure Versioning

# CSRM Payroll System

---

# PURPOSE

Defines:
salary structure history preservation strategy.

---

# IMPORTANT PRINCIPLE

Employee salary changes should preserve:
historical payroll integrity.

---

# VERSIONING STRATEGY

When salary changes:

- old salary structure becomes inactive
- new salary structure becomes active

Old payroll records remain unchanged.

---

# IMPORTANT RULE

Never overwrite historical salary data.

---

# COMMON CHANGE TYPES

- increment
- promotion
- allowance change
- deduction change

---

# FUTURE FEATURES

- effective date calculation
- future salary activation
- salary comparison report
